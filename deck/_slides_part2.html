TAKEAWAYS
18 / 19
收束

三個帶走的重點

01

不是新引擎，是 Claude Code + 殼 + 沙盒

同一顆引擎，外面包 UI、VM、一組新工具與功能。

02

安全的核心 trade-off

炸不到 host，代價是 runtime 只能用 VM 的、碰不到 host 環境。

03

能力邊界決定用法

自包含任務丟 Cowork；要碰 host 的回 Claude Code。
